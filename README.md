## Starting the Development Servers

### Backend

1. Navigate to the backend directory:

   ```bash
   cd backend
   ```

2. Start the backend server:

   ```bash
   npm run dev
   ```

### Frontend

1. Navigate to the frontend directory:

   ```bash
   cd ../frontend
   ```

2. Start the frontend server:

   ```bash
   npm start
   ```
